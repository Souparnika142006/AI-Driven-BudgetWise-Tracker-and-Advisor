# AI-Driven BudgetWise Tracker and Advisor

AI-Driven BudgetWise Tracker is a smart personal finance management system that helps users track income and expenses, set budgets, and monitor savings. It uses the Google Gemini LLM to analyze spending patterns and provide personalized financial insights and recommendations for improving budgeting and reaching savings goals.

## Features

- Track income and expenses
- Create and manage budgets
- Monitor savings goals
- Receive AI-powered financial suggestions

## Run Locally

Prerequisites:
- Node.js

1. Install dependencies:
   ```bash
   npm install
   ```
2. Set your Gemini API key in `.env.local`.
3. Start the app:
   ```bash
   npm run dev
   ```
